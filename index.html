BioSymphony

AI agent tools for biological research.

We build small, readable repositories that help scientists turn biological questions into local workflows an agent can run, explain, and hand back for human judgment.

View GitHub Choose a toolkit

Each toolkit gives a coding agent explicit instructions, examples, and local checks for a specific biological workflow.

The scientist stays close to the work: setting goals, choosing data sources, approving compute, and deciding what is useful.

Shared pattern

Make agent work concrete enough to inspect.

01

Start local.

Run the first demo and the smallest useful checks before growing the workflow.

02

Give the agent a real surface.

Each repo carries instructions, examples, templates, and commands the agent can follow.

03

Keep data where it belongs.

Public examples stay public; private data and larger artifacts stay in operator-owned storage.

04

Grow only when useful.

Move to cloud, HPC, or GPU resources after the local path is clear enough to justify it.

How to begin

Choose one repo and run its first demo.

  1. Pick the toolkit closest to your research question.
  2. Read the repo README.md and AGENTS.md.
  3. Open the repo in your coding agent and ask for the local demo.
