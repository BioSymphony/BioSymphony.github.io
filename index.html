Open research tools

BioSymphony

AI agent tools for biological research.

We build open, readable toolkits that help scientists turn biological questions into agent-guided workflows they can run locally, review, and extend.

View GitHub Choose a toolkit

Each toolkit gives a coding agent a defined place to start: instructions, examples, templates, and local checks for a specific biological workflow.

The scientist stays close to the work: setting goals, choosing data sources, approving compute, and deciding what is useful.

Shared pattern

Start local, then expand with care.

01

Start local.

Run the first demo and the smallest useful checks before adding more scope.

02

Give the agent clear instructions.

Each repo carries instructions, examples, templates, and commands the agent can follow.

03

Keep data where it belongs.

Public examples stay public; private data and larger artifacts stay in operator-owned storage.

04

Expand when the path is clear.

Move to cloud, HPC, or GPU resources after the local path is clear enough to support the next step.

How to begin

Choose one repo and run its first demo.

  1. Choose the toolkit closest to your research question.
  2. Read the repo README.md and AGENTS.md.
  3. Open the repo in your coding agent and ask it to run the local demo.
